Расчет стоимости автомобиля в лизинг

Web-сайт | 2021
Свой собственный проект

Проект создан для расчета стоимости автомобиля в лизинг. При расчете конечной стоимости автомобиля используется разная ставка для каждого банка. С помощью ползунка вы можете выбрать стоимость автомобиля, срок лизинга и первоначальный взнос. Калькулятор способен рассчитать и точно вывести сумму кредита, ежемесячный платеж, рекомендуемый доход и сумма договора лизинга

Технологии: JavaScript / HTML / CSS
